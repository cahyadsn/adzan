INSTALASI

Sebelum melakukan proses instalasi, disarankan Anda melihat terlebih dahulu Kebutuhan Sistem.

Langkah 1: Unzip File
Unzip file hasil download Anda ke komputer lokal, pastikan bahwa struktur direktori yang dibentuk sesuai dengan yang ada dalam file zip. Jika Anda tidak memiliki program bantu untuk zip, Anda dapat mengambil file program Winzip di http://www.winzip.com/ .

Langkah 2: Meng-upload File
Menggunakan program FTP, login ke web server Anda dan upload folder 'adzan' beserta isinya dari komputer lokal ke web root di web server Anda. Biasanya dalam folder public/html atau htdocs, pada direktori yang sama dengan letak homepage Anda. Meski begitu, bisa juga folder web root berbeda letakknya pada server Anda.

Langkah 3: Mengatur File Permissions
ADZAN memerlukan pengubahan beberapa file dan direktori. Jika Anda memakai Unix-based web server, Anda perlu mengatur file permissions - read, write dan execute pada file dan direktori ini. Kebanyakan program FTP memungkinkan Anda untuk mengubah file permissions. Untuk pengguna Windows server, file permissions biasanya telah diatur secara otomatis.

Permissions dinyatakan dalam 3 digit angka, seperti 644 atau 755. Silakan CHMOD* file dan direktori berikut ini:

  • /adzan/config.php : 777 (Owner: Read, Write, Execute; Group: Read, Write, Execute; Others: Read, Write, Execute)
  • /adzan/data/ : 777 (Owner: Read, Write, Execute; Group: Read, Write, Execute; Others: Read, Write, Execute)
  • /adzan/data/america.txt : 777 (Owner: Read, Write, Execute; Group: Read, Write, Execute; Others: Read, Write, Execute)
  • /adzan/data/egypt.txt : 777 (Owner: Read, Write, Execute; Group: Read, Write, Execute; Others: Read, Write, Execute)
  • /adzan/data/indonesia.txt : 777 (Owner: Read, Write, Execute; Group: Read, Write, Execute; Others: Read, Write, Execute)

Unix/Linux File Permissions
)* CHMOD adalah perintah di lingkungan *NIX untuk mengubah file permissions.

Untuk mengatur file permissions dengan program FTP Anda, 'right click' pada file atau direktori, dan pilih perintah sperti 'File Attributes', 'Change File Permissions', atau 'chmod'. Dari sini Anda dapat mengatur 'read, write, execute' permissions untuk 'owner, group, others' melalui checkbox. Bagaimanapun juga, cara termudah untuk mengatur permissions adalah dengan memasukkan angka-angka yang sesuai.

Windows File Permissions
Umumnya, script PHP dapat berjalan di lingkungan Windows tanpa memerlukan pengaturan file permissions. Namun, dalam beberapa kasus, server Anda mungkin tidak di setup untuk memungkinkan script PHP untuk mempunyai 'write access' untuk file atau direktori tersebut. Dan hasilnya, program akan menampilkan pesan kesalahan bahwa program tidak dapat menulis ke file atau direktori tersebut.

Langkah 4: Jalankan Installation Wizard
Mulai saat ini, semuanya menjadi lebih mudah dan cepat karena Adzan web installer akan mengambil alih semua perintah. Arahkan ke URL http://localhost/adzan/ dimana secara langsung web installer akan menjalankan pre-installation check. Pengecekan ini akan memperlihatkan apakah lingkungan sistem yang Anda gunakan sesuai instalasi ADZAN. Jika ditemukan banyak tanda hijau dari hasil test, berarti sistem yang Anda gunakan bagus. Mungkin saja hasilnya akan berbeda tergantung dari konfigurasi sistem yang Anda atur.
