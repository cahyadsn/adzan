INSTALLATION

Before installing, we recommend you to take a look at the System Requirements.

Step 1: Unzip File
Unzip your downloaded zip file onto your local computer, making sure to preserve the directory structure contained in the zip. If you do not have a zip utility, you can download the Winzip program at http://www.winzip.com/ .

Step 2: Uploading Files
Using the FTP program, login to your web server and upload the 'adzan221' folder from your local computer into the web root. This is usually public_html or htdocs folder, the same directory in which your homepage resides. However, the web root folder could be different on your server.

Step 3: Set File Permissions
Adzan requires modification of files in directories. If you are using Unix-based web server, you will need to set the file permissions - read, write and execute on these files and directories. Most FTP programs allow you to change file permissions. As for Windows server users, the appropiate file permissions might have been set.

Permissions are specified by 3 digit number, such as 644 or 755. Please CHMOD* these file and directory accordingly:

  • /adzan/config.php : 777 (Owner: Read, Write, Execute; Group: Read, Write, Execute; Others: Read, Write, Execute)
  • /adzan/data/ : 777 (Owner: Read, Write, Execute; Group: Read, Write, Execute; Others: Read, Write, Execute)
  • /adzan/data/city.txt : 777 (Owner: Read, Write, Execute; Group: Read, Write, Execute; Others: Read, Write, Execute)

Unix File Permissions
)* CHMOD is a command used for changing file permissions.

To set file permissions with your FTP program, right click on a file or directory, and look for command such as 'File Attributes', 'Change File Permissions', or 'chmod'. From here you will be able to set 'read, write, execute' permissions for 'owner, group, others' via checkboxes. Anyway, the easiest way to set the permissions would be insert the appropiate number.

Windows File Permissions
Normally, PHP scripts is able to run on Windows environment without the need to set any file permissions. However, in some cases, your server may not be setup to allow the PHP scripts to have 'write access' to it's file. As a result, the program will prompts you warning message stating that it is not able to write into it's data file.

Step 4: Run Installation Wizard
From now on, everything is going to go lightning fast because the Adzan web installer will be taking over command. Go to the URL http://localhost/adzan/ where the web installer announces itself with a pre-installation check. This check determines whether your environment is suitable for installing Adzan. If there are many green test results, then it is already a good sign. Depending on your configuration there can be differences here.

The web installer takes the configuration settings of Apache, PHP, and the operating system into consideration. On Linux-based systems, attention should be given to writing rights. The web installer should look as shown in the following figure:

Figure 1: Pre-Installation Check

Click on Next to get the announcement of the GNU/GPL license, which you must accept by marking the I Accept the GPL License field. The installation with the web installer takes place in the following three steps:

  • STEP 1 Setting Parameters
  • STEP 2 Directory Paths and Administrator Password
